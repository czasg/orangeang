流程控制

背景​

不同的编程语言通常都具有类似的流程控制结构，但在Go语言中，有一些独特之处。

比如，Go语言没有像其他语言中的 try ... catch ... 异常处理机制，而是采用独有的错误链机制...
再比如，Go语言没有with lock.lock()上下文管理机制，而是采用defer延迟函数机制...
...

本节教程主要介绍Go中常用的流程控制机制。

目的​

学习并了解Go中常用的流程控制机制。

条件语句（if）​

在 Go 中，if-else是常用的条件判断语句，其基本结构如下：

if bool条件 {
  // code
} else if bool条件 {
  // code
} else {
  // code
}
http://localhost:3000
  • 错误值判断
  • 字符串包含关系判断
caller := func() error {
  return nil
}
err := caller()
if err != nil {
  fmt.Printf("error occur：%v", err)
}
if strings.Contains("orange-ang", "cool") {
  fmt.Println("yes~")
} else if strings.Contains("orange-ang", "crazy") {
  fmt.Println("yes~")
} else {
  fmt.Println("no!")
}

循环语句（for）​

在 Go 中，for是常用的循环语句，其基本结构如下：

for 初始值; 条件判断; 循环处理 {
  // code
}
http://localhost:3000
  • 普通循环
  • 无限循环
  • while等价循环
  • 循环切片
for i := 0; i < 10; i++ {
  fmt.Println(i)
}
count := 0
for {
  fmt.Println(count++)
}
count := 0
for count < 100 {
  fmt.Println(count++)
}
arr := []int{10, 20, 30}
for index, value := range arr {
  fmt.Println(index, value)
}

选择语句（switch）​

在 Go 中，switch和select是常用的选择语句，用于根据条件执行不同的代码块。考虑本教程之前没有介绍通道chan，因此暂时不会介绍select。

// switch 语句用于根据不同的条件执行不同的代码块。
switch value {
case value1:
    // 执行与 value 等于 value1 时对应的代码块
case value2:
    // 执行与 value 等于 value2 时对应的代码块
default:
    // 如果以上 case 都不满足，则执行 default 代码块（可选）
}
http://localhost:3000
  • 常规使用
  • fallthrough
package main

import (
    "fmt"
)

func main() {
    value := "value"
    switch value {
    case "value":
        fmt.Println("value")
    case "value1", "value2":
        fmt.Println("value1/value2")
    default:
        fmt.Println("default")
    }
}
package main

import (
    "fmt"
)

func main() {
    value := "value"
    switch value {
    case "value":
        fmt.Println("value")
        fallthrough // 执行下一个 case 分支
    case "value1", "value2":
        fmt.Println("value1/value2")
    default:
        fmt.Println("default")
    }
}
