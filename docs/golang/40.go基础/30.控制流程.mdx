编程是一项极具逻辑思维的工作。

流程控制则是编程的基础。开发者通过流程控制编写特定的代码，从而控制程序的执行顺序与逻辑。

通过条件判断、循环处理以及中断等基本的流程控制手段，开发者能够实现更为复杂的业务逻辑和功能。










## 条件语句（if）

在 Go 中，`if-else`是常用的条件判断语句，其基本结构如下：

```go
if bool条件 {
  // code
} else if bool条件 {
  // code
} else {
  // code
}
```

## 循环语句（for）

在 Go 中，`for`是常用的循环语句，其基本结构如下：

```go
for 初始值; 条件判断; 循环处理 {
  // code
}
```

## 选择语句（switch）

在 Go 中，`switch`和`select`是常用的选择语句，用于根据条件执行不同的代码块。考虑本教程之前没有介绍通道`chan`，因此暂时不会介绍`select`。

```go
// switch 语句用于根据不同的条件执行不同的代码块。
switch value {
case value1:
    // 执行与 value 等于 value1 时对应的代码块
case value2:
    // 执行与 value 等于 value2 时对应的代码块
default:
    // 如果以上 case 都不满足，则执行 default 代码块（可选）
}
```
