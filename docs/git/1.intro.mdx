## 背景

目前网上 git 教程非常多，我找过也看过不少相关的教程。和大家分享下其中我认为写的比较好的教程：

- [廖雪峰的官方网站](https://www.liaoxuefeng.com/wiki/896043488029600)

对于 git，我其实从未深度学习过，或者说我暂时没有遇到过什么问题，需要我研究 git 底层原理才能解决。
对于 git，我仅仅是把它当做一个方便的工具使用~

因此，本教程不会深入探讨 git 的底层原理和使用方法，而是面向团队的 git 教程。
教程内容将更加注重实践操作，旨在为团队提供 git 最佳实践指南。


## 目的

    - 常用 rebase

- 实战演练/冲突解决


## 安装

参考[官方文档](https://git-scm.com/book/zh/v2/%E8%B5%B7%E6%AD%A5-%E5%AE%89%E8%A3%85-Git)

### linux安装

在 linux 中安装 git 比较简单，甚至大部分系统都自带 git 指令。此处主要介绍 ubuntu/centos 系统下 git 安装方法。

```shell title="ubuntu系统"
sudo apt update
sudo apt install git
```

```shell title="centos系统"
sudo yum install git
```

### window安装

访问 https://git-scm.com/download/win 下载安装即可
